## Overview
</div>

- **Playtesting build** - This beta is for testing and giving bug reports
- **Do not report placeholders** - Those are dev tools for future updates  
- **More planets** - Multiple planets added, 1 fully playable for now
- **Terra** - Nearly no changes, no custom terrain yet
- **Solo dev** - Made entirely by 24lowie

<br>

<div align = center>
## Download & Install
</div>

1. **Copy this folder** and add it to the Mindustry `mods` folder
2. **Use Mindustry build 154+** - Older versions do not work  
3. **Backup saves** - I recommend making a save of your world before playing

<br>
